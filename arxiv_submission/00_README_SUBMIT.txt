arXiv submission bundle
=======================
Compile order: pdflatex paper.tex  (x2 for refs)
Files:
  paper.tex        main source (\input{numbers.tex}, \input{table_main.tex})
  numbers.tex      auto-generated numeric macros (from results/)
  table_main.tex   auto-generated main results table
  figures/*.pdf    vector figures
Recommended arXiv category: cs.CL (primary), cs.IR / cs.DC (cross-list).
All numbers are reproducible offline via the harness (see project README).
